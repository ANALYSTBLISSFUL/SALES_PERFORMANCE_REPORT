# SALES_PERFORMANCE_REPORT

### Project Overview

This project is an interactive Excel dashboard developed to analyze sales performance across products, cities, sales representatives, customer types, and sales channels.

### Objective

The goal was to transform raw sales data into meaningful business insights that support management decision-making.

### Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

### Key Metrics

- Revenue
- Profit
- Profit Margin
- Cost of Goods Sold (COGS)

### Dashboard Features

- Revenue by Sales Representative
- Profit by Product
- COGS by City
- Monthly Profit by Customer Type
- Interactive filters for:
  - Region
  - Product Category
  - Sales Channel
  - Customer Type
  - City
  - Sales Representative

### Business Insights

- Identified the highest-performing sales representatives.
- Compared profitability across products.
- Evaluated COGS across different cities.
- Analyzed customer profitability trends.

### Skills Demonstrated

- Data Cleaning
- Excel Dashboard Development
- Business Analysis
- Data Visualization
- KPI Reporting
- Storytelling with Data
